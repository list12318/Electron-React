## Getting Started

### Installation

1. Clone the repo

```sh
git clone https://github.com/list12318/Electron-React.git
```

2. Install packages

```sh
cd Electron-React
npm install

cd app
npm install
```

3. Run using command:

```sh
cd Electron-React
npm start

cd app
npm run start-electron
```

## Build

```sh
cd Electron-React
npm run build
```

1. Windows

```sh
cd app
npm run build:windows
```

2. Linux

```sh
cd app
npm run build:linux
```
